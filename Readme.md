Nom et Description
	
	Name: ImagesBridge
Description: 
	Allows selecting and opening images in multiple external editors using dedicated buttons.

Informations Techniques
	Blender Version: Compatible with Blender 4.3.1 and above.
	Category: Image
	Version: 1.0.27
	Author: lesergei3d

Fonctionnalités Principales

1. Gestion des éditeurs externes :
	Ajoutez plusieurs éditeurs d'images avec des noms personnalisés et leurs chemins.
	Réorganisez ou supprimez les éditeurs configurés.

2. Ouverture rapide :
	Ouvrez l'image active directement dans l'éditeur externe sélectionné.
	Fonctionne depuis l'Éditeur d'Images, la Vue 3D, ou les Propriétés de Matériau.

3. Sauvegarde simplifiée :
	Sauvegarde automatique ou affichage d'une boîte de dialogue si l'image n'est pas encore sauvegardée.
	
Guide d'Utilisation

1. Configuration :
	Accédez aux préférences de l'addon (Édition > Préférences > Add-ons).
	Ajoutez un éditeur avec le bouton "Add Editor".
	Renseignez le nom et le chemin de l'éditeur.

2. Utilisation :

	Sélectionnez une image active dans Blender (Vue 3D ou Éditeur d'Images).
	Utilisez les boutons affichés dans l'interface de l'addon pour ouvrir ou sauvegarder l'image.

3. Astuce :

	Assurez-vous que l'image active est sauvegardée avant de l'envoyer à un éditeur externe.

Remarques Techniques
	
	Compatibilité : 
	Vérifiez que les chemins des éditeurs configurés sont valides.
	
	Limites : 
	L'ouverture d'images non sauvegardées déclenche une boîte de dialogue pour choisir un emplacement de fichier.

Main Features

1. Management of external editors:
  Add multiple image editors with custom names and their paths.
  Reorganize or delete configured editors.

2. Quick opening:
  Open the active image directly in the selected external editor.
  Works from Image Editor, 3D View or Material Properties.

3. Simplified backup:
  Auto-save or display a dialog box if the image is not yet saved.

User Guide

1. Configuration:
  Go to the addon preferences (Edit > Preferences > Add-ons).
  Add an editor with the "Add Editor" button.
  Fill in the name and path of the editor.

2. Use:

  Select an active image in Blender (3D View or Image Editor).
  Use the buttons displayed in the addon interface to open or save the image.

3. Tip:

  Make sure the active image is saved before sending it to an external editor.

Technical Notes

  Compatibility:
  Check that the configured editor paths are valid.

  Boundaries :
  Opening unsaved images triggers a dialog to choose a file location.
